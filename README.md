# 项目说明

本项目用于在云端完成深度学习模型训练，并基于训练结果进行可视化与大气参数分析。

---

## 文件说明

| 文件名 | 作用 |
|------|------|
| `Main_trainDL.py` | 模型训练与评估入口文件，**仅导入第三方库，不依赖其他自定义 `.py` 文件** |
| `Main_PlotFig.py` | 根据训练结果绘制各类图像 |
| `Main_Active_atmospheric.py` | 基于训练结果进行大气参数分析，输出结果为 `integrated_active_atmospheric_features.csv` |

---
